# Keyboard Shortcuts

## File

| Shortcut | Action |
|----------|--------|
| Ctrl+N | New report |
| Ctrl+O | Open file |
| Ctrl+S | Save |

## Edit

| Shortcut | Action |
|----------|--------|
| Ctrl+Z | Undo |
| Ctrl+Y | Redo |
| Ctrl+C | Copy |
| Ctrl+X | Cut |
| Ctrl+V | Paste (smart offset +4px per paste) |
| Ctrl+D | Duplicate |
| Del | Delete selected |
| Ctrl+A | Select all |
| Ctrl+G | Group |
| Ctrl+Shift+G | Ungroup |

## Navigation

| Shortcut | Action |
|----------|--------|
| Arrow keys | Nudge element 1px |
| Shift+Arrow | Nudge 8px |
| Ctrl+Wheel | Zoom in/out |
| Space+Drag | Pan canvas |
| Ctrl+0 | Zoom 100% |
| Ctrl++ | Zoom in |
| Ctrl+- | Zoom out |

## View / Modes

| Shortcut | Action |
|----------|--------|
| F5 | Preview report |
| Esc | Return to Design / Select tool |
| Ctrl+F | Formula Workshop |
| F9 | SQL Expression Editor |
| Ctrl+Shift+O | Object Explorer |
| Ctrl+Shift+F | Formula Debugger |
| Ctrl+R | Repository Explorer |

## Z-order

| Shortcut | Action |
|----------|--------|
| Ctrl+] | Bring forward |
| Ctrl+[ | Send backward |

## Tools (when not editing)

| Key | Tool |
|-----|------|
| V | Select tool |
| T | Text object |
| F | Field |
| L | Line |
| R | Rectangle |
| I | Image |

## Alt accelerators (menu bar)

`Alt` activates the menu bar. Use arrow keys to navigate. Letter underlines trigger items.
